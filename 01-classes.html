Classes

A CSS class is an identifier of an element which can be used multiple times on a page.

Classes in schools are periods (first and second period, etc.) so remember to use a . when you want to refernce a clas in CSS.

  • Classes start with a . in the selector (.anyting-you-want-to-name-it)
  • The same class name can be used more than once within the same file.
  • The class keyword refers to the class name that you set in HTML
